Esse é um código simples feito com o intutio de praticar conceitos básicos de programação servindo como preparação para uma prova de monitoria da disciplina Funcamentos de Programação.
Com base nesta disciplina que cursei, utilizei apenas conceitos vistos nela, com exceção de algumas funções para algumas funcionalidades como a rand(). Não usei estruturas de dados mais complexas nem alocação dinâmica.

Porém, como meu primeiro commit será já depois de fazer a prova, pretendo tentar otimizar o código com esses conceitos para praticar.

Baseei no Desafio 22 do livro [Fundamentos da Programação de Computadores: Algoritmos, Pascal, C, C++ e Java](https://www.amazon.com.br/Fundamentos-programa%C3%A7%C3%A3o-computadores-algoritmos-pascal/dp/8564574160), não segui necessáriamente a risca as especificações.
